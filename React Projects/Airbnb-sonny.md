# Command line installs

- `npx create-next-app -e with-tailwindcss my-project'
- `npm install @heroicons/react`
- `npm i tailwind-scrollbar-hide`
- `npm install --save react-date-range` & `npm install --save react date-fns`
- `npm i @badrap/bar-of-progress`

# Deploy in Vercel
- Push the code in Github repo.
- Every time you push into github, vercel redeploys the application

# Mapbox integration
- Go to mapbox, create account and copy the access token
- `npm install --save react-map-gl`
- `npm install geolib`
